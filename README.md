# NEXORA

> Think. Search. Build.

NEXORA is an AI-powered workspace built with FastAPI and Next.js, designed to combine conversational AI, real-time web search, intelligent coding assistance, and future AI agent capabilities into a single modern platform.

---

## Tech Stack

### Frontend

- Next.js
- React
- Tailwind CSS
- TypeScript
- Framer Motion

### Backend

- FastAPI
- Python
- Groq API

---

## Current Features

- AI Chat
- Streaming Responses
- Web Search
- Markdown Rendering
- Syntax Highlighting
- Code Copy
- Smart Composer
- Smart Scrolling
